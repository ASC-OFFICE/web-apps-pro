Вкладка Файл

Вкладка Файл позволяет выполнить некоторые базовые операции с текущим файлом.

С помощью этой вкладки вы можете выполнить следующие действия:

  • сохранить текущий файл в текущем формате и местоположении с помощью опции Сохранить, сохранить файл под другим именем, в другом местоположении или в другом формате с помощью опции Сохранить как, распечатать файл,
  • просмотреть общие сведения об электронной таблице,
  • управлять правами доступа (после подключения к облачной версии),
  • открыть дополнительные параметры редактора,
  • открыть окно Проводника, чтобы выбрать другой файл.
